Открытие главного экрана приложения MegaPlan
java
Intent intent = new Intent(Intent.ACTION_MAIN);
intent.addCategory(Intent.CATEGORY_LAUNCHER);
intent.setPackage("ru.megaplan.megaplan3app"); // Указываем пакет MegaPlan
startActivity(intent);

====

2. Открытие карточки по URL
Если вы хотите открыть конкретную карточку в приложении MegaPlan, используйте следующий код:
java
Intent intent = new Intent(Intent.ACTION_VIEW);
intent.setData(Uri.parse("https://your.megaplan.url/card/12345")); // Замените на нужный URL карточки
intent.addCategory(Intent.CATEGORY_BROWSABLE);
startActivity(intent);

====
3. Открытие через схему "megaplan"
Для открытия через схему megaplan, используйте следующий код:
java
Intent intent = new Intent(Intent.ACTION_VIEW);
intent.setData(Uri.parse("megaplan://some/path")); // Замените на нужный путь
startActivity(intent);
